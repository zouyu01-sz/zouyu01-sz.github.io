安装nginx

  1. 基础运行环境

2：切换目录：cd /usr/local
3：创建文件夹：mkdir nginx 并 cd nginx
4：下载nginx：wget http://nginx.org/download/nginx-1.13.7.tar.gz
5：解压：tar -xvf nginx-1.13.7.tar.gz
6：切换目录：cd nginx-1.13.7
7：make install
8：查看nginx进程 ps -ef | grep nginx

Nginx指令无法使用，问题如下

  1. vim /etc/profile 进入文件进行编辑
    ‘/usr/local/nginx/sbin’ 代表nginx的安装位置
  2. 新增以下PATH
  3. source /etc/profile 让配置文件重新生效一下
